Native Client Bridge (e.g. Fladder)

Clients that can't run the injected browser sync script (e.g. Fladder on Android TV) can still host a watch party: pick a currently-playing session below to bridge its playback into a new OpenWatchParty room, with that session as host. Anyone using Jellyfin Web joins the resulting room from their own room list, exactly as they would any other room — this page doesn't join anyone automatically.
